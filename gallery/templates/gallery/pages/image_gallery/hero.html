DJANGO GALLERY — Custom Photo Gallery

 This is a custom photo gallery developed to allow a user to upload a single image or a bulk set of images. Upon upload images will be resized if they are larger than 1200px wide to allow for a maximum of only 1200px wide. This is to ensure the gallery does not become too heavy on assets when a user visits this page. Additionally, images are compressed and optimized to 60% quality and if they are not of the approved image file extensions, they will be converted to a standard JPG. Other accepted formats such as PNG images are optimized using lossless optimization. Color modes such as RGBA/P are converted to a standard RGB. This is also to ensure that the physical file size is not too large and can reduce them down to the low KB range. If we have a gallery with 200 images all around 20MB in size, that would mean our page when loading will want to load a minimum of almost 4GB of data. That is far too much for any web page and especially mobile devices. Additionally, this would relate to increased cost for server data storage in maintaining the cost of your website.
